Installation

Using Composer

Composer is a package manager for PHP. It's the best way to install and manage your project dependencies.

Install Composer: 

curl -s https://getcomposer.org/installer | php

Create a project using the skeleton application: 

$ php composer.phar create-project atomik/skeleton .
